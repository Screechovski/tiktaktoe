
# Tik tak toe game

Tic-tac-toe game project `alpha`


## Installation

### Run project

Install server dependencies
```bash
npm i;
```

Run server build
```bash
npx tsc -w;
```

Run server
```bash
npx nodemon dist/main.js;
```

Install client dependencies
```bash
cd client;
npm i;
```

Run client
```bash
npm run dev;
```

Open [http://localhost:5173](http://localhost:5173/) link
